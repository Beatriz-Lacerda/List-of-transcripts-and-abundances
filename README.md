# List-of-transcripts-and-abundances
Table S1. List of transcripts assembled and abundances estimated (transcripts only for simplicity)

Columns are as follows:
1.	seqname: name of the chromosome or scaffold (Ensembl)
2.	source: name of the program that generated this feature
3.	feature: feature type name 
4.	start: Start position of the feature
5.	end: End position of the feature
6.	score: A floating point value
7.	strand: defined as + (forward) or - (reverse)
8.	frame: One of '0', '1' or '2'. '0' indicates that the first base of the feature is the first base of a codon, '1' that the second base is the first base of a codon, etc
9.	attribute: A semicolon-separated list of tag-value pairs, providing additional information about each feature
